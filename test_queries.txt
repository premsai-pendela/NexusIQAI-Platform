## **🐛 KNOWN ISSUES TO WATCH FOR**

1. **Web Freshness**: A failed live refresh may show clearly labeled stale cache for up to 7 days; verify its capture time is visible
2. **Quota Exhaustion**: If Gemini Flash fails, should fallback to Groq
3. **Empty Results**: Sample Web data is disabled by default and must never appear as successful live evidence
4. **Date Parsing**: Comparison queries need careful date extraction
5. **Large Tables**: SQL results >100 rows might slow UI
